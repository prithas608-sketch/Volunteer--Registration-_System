# Volunteer Registration System

## Project Overview

The Volunteer Registration System is a full-stack web application developed to manage volunteer registrations efficiently. The system allows users to submit their personal details through an online registration form, and the data is securely stored in a MongoDB database.

## Features

* Volunteer registration form
* User-friendly interface
* Real-time data submission
* MongoDB database integration
* Backend API using Express.js
* Secure data storage and retrieval

## Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* Express.js
* MongoDB Atlas

## How It Works

Users fill out the registration form with details such as name, email, phone number, age, address, skills, and availability. When the form is submitted, the data is sent to the backend server through REST APIs and stored in MongoDB Atlas.
